🐍 Snake Game

🎮 About the Project

Snake Game is a classic arcade-style web game built using modern web technologies. It features smooth gameplay, responsive controls, and a clean UI. This project demonstrates strong fundamentals of JavaScript, DOM manipulation, and game logic.

---

🚀 Features

- 🎯 Real-time snake movement
- 🍎 Food generation & score tracking
- ⚡ Smooth controls (keyboard-based)
- 🔁 Game over & restart functionality
- 📱 Responsive design

---

🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (Vanilla JS)

---

▶️ How to Run the Project

1. Clone the repository:

git clone https://github.com/your-username/snake-game.git

2. Navigate to project folder:

cd snake-game

3. Open "index.html" in your browser
   (OR use Live Server in VS Code)

---

🎮 How to Play

- Use arrow keys (⬆️ ⬇️ ⬅️ ➡️) to move the snake
- Eat food to grow and increase score
- Avoid hitting walls or yourself
- Game ends on collision
