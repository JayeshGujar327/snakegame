🎮 Snake Game

Classic multiplayer snake experience built using JavaScript & Socket.io

👨‍💻 Credits

Created by Jayesh Gujar

Frontend: HTML, CSS, JavaScript

Backend: Node.js + Socket.io

🎯 How to Play

Control your snake, eat food, and survive longer than others!

⬆️ Arrow Up → Move Up

⬇️ Arrow Down → Move Down

⬅️ Arrow Left → Move Left

➡️ Arrow Right → Move Right

⚠️ Game Rules

🍎 Eat food to grow your snake

💥 Avoid hitting walls

☠️ Avoid hitting yourself or other players

🏆 Last player alive wins
